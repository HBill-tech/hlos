## 理解不同进制下的规律不变性

### 1.概述

* **定义 $`x_{(n)}`$ 表示数 $`x`$ 在 $`n`$ 进制下的表示.**

如果有一个方程
```math
a_{(10)} = b_{(10)}  + c_{(10)} 
```

那么必然有
```math
a_{(2)} = b_{(2)}  + c_{(2)}
```

可以推广为

```math
a_{(n)} = b_{(n)}  + c_{(n)}
```

### 2.举例：根据 `BCD` 码求它实际二进制数值的方法

#### 什么是 `BCD` 码

* `BCD` 码是一种用4位二进制数表示1位十进制数的编码方式.
* `BCD` 码能表示的十进制整数范围是 0 ~ 99.

#### 定义

* $`value`$：`BCD` 码表示的数
* $`BCD03`$：`BCD` 码低四位的数值
* $`BCD47`$：`BCD` 码高四位的数值
* $`K`$：常数，在十进制下等于10

显然得到十进制下 $`value`$ 的计算公式（记作公式1）
```math
value_{(10)} = BCD03_{(10)} + BCD47_{(10)} \times K_{(10)}
```

于是得到 $`value`$ 在二进制下的计算公式为（记作公式2）
```math
value_{(2)} = BCD03_{(2)} + BCD47_{(2)} \times K_{(2)}
```

**说明：**
- 这里的 $`value_{(10)}`$ 是 `BCD` 码所表示的十进制数.
- 这里的 $`value_{(2)}`$ 是 $`value_{(10)}`$ 的二进制表示，即 `BCD` 码表示的二进制数.
- 计算机恰好可以提供 `公式2` 中计算 $`value_{(2)}`$ 所需要的每一项数字以及运算，因此在程序中使用该公式计算 `BCD` 码表示的二进制数.

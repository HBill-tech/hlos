## 理解不同进制下的规律不变性

### 1.概述
如果有一个方程
```math
a_{(10)} = b_{(10)}  + c_{(10)} 
```

那么必然有
```math
a_{(2)} = b_{(2)}  + c_{(2)}
```

可以推广为

```math
a_{(n)} = b_{(n)}  + c_{(n)}
```

### 2.举例：根据BCD码求它实际二进制数值的方法

#### 什么是BCD码

* BCD码是一种用4位二进制数表示1位十进制数的编码方式.
* BCD码能表示的十进制整数范围是 0 ~ 99.

#### 定义

* $`value`$：BCD码的数值
* $`BCD03`$：BCD码低四位的数值
* $`BCD47`$：BCD码高四位的数值
* $`K`$：常数，在十进制下等于10

显然得到
```math
value_{(10)} = BCD03_{(10)} + BCD47_{(10)} \times K_{(10)}
```

于是可得
```math
value_{(2)} = BCD03_{(2)} + BCD47_{(2)} \times K_{(2)}
```
